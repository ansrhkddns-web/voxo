# VOXO 고도화 로드맵

작성일: 2026-03-10

## 1. 문서 목적

이 문서는 VOXO를 어떤 순서로 고도화할지 정리한 실행 기준서입니다.

핵심은 단순합니다.

- 무엇을 먼저 고쳐야 하는지
- 어떤 기능을 다음으로 붙여야 하는지
- 지금 당장 손대면 효과가 큰 일이 무엇인지

## 2. 현재 판단

현재 VOXO는 기본 골격은 충분히 올라와 있습니다.

- 공개 사이트 동작 가능
- 관리자 CMS 진입 가능
- AI 초안 생성 가능
- Supabase 연동 가능

하지만 실제 운영 관점에서는 아직 아래가 부족합니다.

- 관리자 문구와 일부 화면 완성도
- 설정 반영 마감도
- 뉴스레터 실발송
- 이미지/SEO/공유 기능 고도화

## 3. 고도화 방향

### 방향 1. 오류를 먼저 없앤다

신규 기능보다 먼저, 기존 기능이 안정적으로 돌아가도록 정리합니다.

이번 작업으로 여기까지는 확보했습니다.

- 빌드 통과
- 린트 오류 0건
- AI 생성 로직 공통화

아직 남은 품질 항목:

- 이미지 최적화 경고 4건
- 관리자 설정/번역 문구 정리

### 방향 2. 운영 기능을 완성한다

실제 서비스를 돌릴 때 필요한 기능을 먼저 끝냅니다.

핵심 항목:

- 뉴스레터 실발송
- 설정 페이지 실사용 정리
- 관리자 운영 흐름 단순화

### 방향 3. 사용자 경험을 높인다

운영 기능이 안정화되면 공개 사이트 경험을 끌어올립니다.

핵심 항목:

- 모바일 UX
- 이미지 최적화
- 공유 기능
- 관련 글 추천
- SEO 강화

## 4. 단계별 로드맵

### Phase 1. 안정화

목표:
배포 가능한 기본 상태 확보

현재 상태:
거의 완료

완료된 항목:

- AI 초안 생성 공통 로직 분리
- 관리자 AI Desk 재정리
- 타입 오류 정리
- 빌드 통과

남은 항목:

- 관리자 설정/번역 문구 복구
- 이미지 최적화 경고 정리

### Phase 2. 운영 완성

목표:
관리자 화면만으로 운영 가능하게 만들기

주요 작업:

- 뉴스레터 발송 서비스 연결
- 발송 결과 처리
- 설정값 실제 반영 강화
- 운영 화면 피드백 정리

### Phase 3. 콘텐츠 제작 고도화

목표:
글 제작과 발행 생산성 향상

주요 작업:

- 에디터 보강
- 미리보기 강화
- SEO 입력/자동화 개선
- 태그/카테고리 흐름 보강

### Phase 4. 공개 UX 고도화

목표:
방문자 경험 개선

주요 작업:

- 모바일 UI 보강
- 검색 UX 정리
- 공유 기능 연결
- 이미지 최적화

### Phase 5. AI/데이터 확장

목표:
VOXO만의 운영 보조 기능 강화

주요 작업:

- AI 생성 이력
- 재시도/실패 복구
- Spotify 캐시
- 관리자 분석 기능

## 5. 오류 수정 계획

### 최우선

- 관리자 설정 페이지 깨진 문구 정리
- 관리자 공통 번역 문자열 정리
- 이미지 최적화 경고 정리

### 다음 단계

- `middleware` 경고 대응
- SEO 메타데이터 정교화
- 공유 기능 실동작 연결

## 6. 기능 추가 계획

### 운영 기능

- 뉴스레터 테스트 발송
- 뉴스레터 실제 발송
- 발송 이력 저장
- 구독자 내보내기

### 콘텐츠 기능

- 관련 글 추천
- 작성 보조 기능
- 포스트 미리보기
- 에디터 품질 개선

### 공개 사이트 기능

- 공유 링크 복사
- 모바일 메뉴 개선
- 검색 결과 고도화
- 빈 상태/오류 상태 화면 개선

## 7. 지금 바로 이어갈 추천 작업

가장 효율적인 다음 순서는 아래와 같습니다.

1. `src/app/admin/settings/page.tsx` 문구/인코딩 정리
2. 관리자 공통 번역 정리
3. 뉴스레터 실발송 서비스 연결
4. `<img>`를 `next/image`로 전환
5. 공유 기능과 SEO 고도화

## 8. 한 줄 결론

지금 VOXO는 "기능 추가"보다 "운영 가능한 수준으로 마감"하는 것이 더 중요합니다.

그래서 다음 개발은 아래 흐름으로 가는 것이 가장 안전합니다.

문구 정리 -> 운영 기능 완성 -> 공개 UX 개선 -> AI/분석 고도화
