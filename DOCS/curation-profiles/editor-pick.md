# Editor Pick

## 언제 쓰는가
- 독자에게 “이건 지금 꼭 들어보세요”라고 추천하고 싶을 때
- 접근성 있고 읽기 쉬운 추천형 큐레이션에 적합

## 글의 목표
- 낯선 곡도 한 번 틀어보고 싶게 만든다.
- 이미 아는 곡은 다시 꺼내 듣고 싶게 만든다.

## 글 구조
1. 왜 지금 이 곡을 추천하는지 한 문장으로 훅 만들기
2. 어떤 순간, 어떤 기분에 어울리는 곡인지 제시
3. 기억에 남는 사운드 포인트 설명
4. 가수 맥락을 짧고 명확하게 연결
5. 마지막에 청취를 부르는 마감

## 문체 방향
- 따뜻하고 설득력 있어야 한다.
- 과장된 홍보문처럼 보이면 안 된다.
- 설명보다 권유의 리듬이 조금 더 살아 있어야 한다.
