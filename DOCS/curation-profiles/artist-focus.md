# Artist Focus

## 언제 쓰는가
- 곡 하나를 출발점으로 가수 전체를 보여주고 싶을 때
- 특정 시대, 커리어, 스타일 변화를 함께 짚고 싶을 때

## 글의 목표
- 이 노래가 가수의 어떤 얼굴을 보여주는지 설명한다.
- 독자가 이 곡을 통해 아티스트 전체 세계관을 이해하게 만든다.

## 글 구조
1. 왜 이 곡이 가수를 이해하는 입구인지
2. 해당 시기의 커리어 배경
3. 곡 안에서 드러나는 미학과 정체성
4. 사운드 분석과 아티스트 해석 연결
5. 커리어 전체 안에서의 의미 정리

## 문체 방향
- 음악 분석과 인물 해설의 균형이 중요하다.
- 곡만 보지 말고, 가수 전체 흐름으로 시야를 확장한다.
