# VOXO - 음악 큐레이팅 웹매거진 고도화 제안서

본 문서는 VOXO를 단순한 리딩(Reading)용 웹매거진을 넘어, 사용자 맞춤형 음악 큐레이션 및 에디토리얼 특화 플랫폼으로 발전시키기 위한 실질적 기능 고도화 아이디어 모음입니다.

---

## 1. � 큐레이션 및 개인화 경험 (Curation & Exploration)
*   **에디터 & 게스트 큐레이터 플레이리스트 (Curated Playlists)**
    *   **개요**: VOXO 에디터 또는 게스트 아티스트가 직접 선곡한 테마별 플레이리스트(예: "심야의 코딩을 위한 딥 테크노", "이번 주말 베를린 클럽 무드")를 정기 발행.
    *   **기능**: 기사 내에 스포티파이 멀티 트랙 엠베드를 제공하고, 클릭 한 번으로 유저의 스포티파이 계정에 해당 플레이리스트를 저장(`Save to Spotify`)하는 기능 연동.
*   **감각적 무드 태그 기반 정밀 탐색 (Deep Tag Browsing)**
    *   **개요**: 기존의 딱딱한 장르(Electronic, Rock 등) 구분을 넘어, `#몽환적인`, `#BPM130+`, `#새벽3시`, `#아날로그신스` 등 세밀한 무드/사운드 태그 시스템 도입.
    *   **기능**: 유저가 특정 태그를 결합하여 탐색하면(예: `#Techno` + `#우울한`), 그에 맞는 기사와 트랙만을 필터링하여 보여주는 고도화된 디깅(Digging) 경험 제공.
*   **'Weekly Drops' (금주의 릴리즈 캘린더)**
    *   **개요**: 매주 쏟아지는 언더그라운드 신보 중 VOXO가 엄선한 트랙들의 발매 일정을 보여주는 인터랙티브 달력.
    *   **기능**: 기대되는 발매일에 '알림 설정'을 구동해, 발매 당일 PWA 기능이나 이메일 리포트로 푸시 알림 전달. 스포티파이 Pre-save 링크 연동.

## 2. 🎧 상호작용형 오디오 플레이어 (Interactive Audio)
*   **글로벌 논스톱 연속 재생 (Persistent Player)**
    *   **개요**: 유저가 기사를 읽다가 메인 페이지로 나가거나 다른 아티클로 이동해도, 하단의 스포티파이 재생 바에서 흘러나오는 음악이 끊기지 않고 계속 이어지는 전역(Global) 네이티브 뷰 레벨 플레이어. 큐레이팅 서비스의 필수 요소.
*   **타임스탬프 리뷰 (Timestamped Annotations)**
    *   **개요**: 트랙 리뷰 기사에서 에디터가 특정 곡의 구간(예: `[02:15] 터지는 아날로그 킥 사운드`)을 언급할 때, 해당 텍스트를 클릭하면 플레이어가 즉시 그 시간대로 점프하여 재생하는 기능. (음악을 글로만 읽지 않고, 바로 함께 듣게 하는 경험)

---

## 3. ✍️ 콘텐츠 및 에디토리얼 포맷 고도화 (이전 채택안)
*   **기사별 특화 레이아웃 템플릿 지원**
    *   *리뷰(Reviews)*: 트랙 메타데이터, 감각적인 별점/수치화 위젯, 'Track Highlights' 섹션이 포함된 템플릿.
    *   *인터뷰(Interviews)*: 스크롤 시 아티스트의 고화질 화보가 한쪽에 고정(Sticky)되고, 질문과 답변이 드라마틱한 타이포그래피 연출로 스크롤되는 스플릿 스크린 방식.
    *   *뉴스(News)*: 속보성에 맞춘 짧고 인스타그래머블한 시각적 레이아웃.
*   **무한 스크롤 & 꼬리물기 기사 (Infinite Scroll Flow)**
    *   기사를 전부 다 읽으면, 클릭할 필요 없이 현재 기사의 태그와 연관된 '다음 추천 큐레이션 기사'가 부드럽게 이어서 밑에서부터 등장. 유저의 체류 시간 극대화.
*   **DB 연동형 동적 뉴스 티커 (Dynamic Marquee)**
    *   메인 페이지 상단에 래스터로 흐르는 긴급 텍스트(Marquee)를 하드코딩이 아닌 Supabase DB와 연동하여, 관리자가 실시간으로 핫 릴리즈나 라인업 소식을 띄울 수 있도록 개편.

## 4. 🛠️ 관리자 자동화 및 테크 기반 고도화 (이전 채택안)
*   **VOXO 전용 CMS 에디터 대시보드 (Admin Panel)**
    *   개발자가 코드를 만지지 않고 에디터들이 비공개 `/admin` 페이지에 로그인하여, WYSIWYG 하이브리드 마크다운 포맷으로 기사 작성, 초안 저장, 예약 발행 등을 할 수 있는 백엔드 인프라(Supabase 연동). 파일 자동 리사이징 업로드 지원.
*   **스포티파이 메타데이터 자동 추출기 (Auto-Fetcher)**
    *   에디터가 CMS 툴에 스포티파이 곡/앨범 URL만 복사해 붙여넣으면, 서버에서 스포티파이 Open API를 호출하여 아티스트명, 앨범 커버 초고화질 이미지, 발매일, 장르/템포 등을 자동으로 긁어와 기사의 메타 박스를 1초 만에 채워주는 단순반복 업무 자동화 로직.
*   **PWA (프로그레시브 웹 앱) 인스톨러 지원**
    *   유저가 VOXO 웹매거진을 모바일 홈 화면에 앱처럼 아이콘으로 설치할 수 있게 지원. 네이티브 앱 스토어 심사 없이도 전체화면(Full-screen) 구동 및 오프라인 일부 캐싱 기능을 제공해 진짜 "음악 큐레이션 스타트업 앱"처럼 작동.
